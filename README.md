# Registration
## 一级组织报名系统
该系统用于给一级组织报名用。

新生可以在此系统上进行报名。
  
后台管理员的各管理关系：
  
超级管理员 > 一级组织管理员 > 部门管理员
  
***
  
### 超级管理员模块
1. 学院管理
2. 组织管理
3. 组织账号管理
4. 报名信息
  
### 一级组织管理员模块
1. 部门管理
2. 部门账号管理
3. 报名信息
  
### 部门管理员
1. 报名信息
  
***

#### 后端
